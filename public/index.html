Everything You Need

A complete key management stack built for reliability and security.

🔒

HWID Binding

Keys are automatically locked to the first hardware ID that redeems them. No sharing, no leaking — one key, one machine.

⏱️

Flexible Expiry

Issue keys for 1 day, 7 days, 30 days, or lifetime. Expiry is enforced server-side on every authentication request.

🛡️

Encrypted Auth Tokens

Successful authentications return an HMAC-SHA256 token derived from the key and HWID — impossible to forge without the server secret.

👥

Reseller System

Grant resellers a balance of keys to distribute. Each reseller can generate keys independently without admin involvement.

🔄

HWID Reset

Admins can reset the hardware lock on any key, letting a user migrate to a new machine without losing their license.

❌

Instant Revocation

Revoke any key immediately from the admin panel. Revoked keys are rejected on the next authentication attempt — no delays.
